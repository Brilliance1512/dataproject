Курсовой проект по курсу 'Специализация Data Science' от SkillFactory

Суть проекта: разработать модель, 
которая будет предсказывать стоимость домов, 
основываясь на истории предложений.
Проект благоприятно скажется на эффективности
и скорости работы риелторов.

Данные для проекта:
https://drive.google.com/file/d/1An8LtTSdv6pB5nSpNBwH1Njcwh2SpYW6/view

Описание данных проекта(data.csv):
status - Текущий статус
private pool - Наличие собственного бассейна
propertyType - Тип жилья
street - Улица
baths - Количество душевых
homeFacts - Факты о доме
fireplace - Наличие камина
city - Город
schools - Список ближайших школ
sqft - Жилая площадь (квадратные футы)
zipcode - Индекс
beds - Количество спален в доме/квартире
state - Штат
stories - Количество этажей в доме
mls-id - id для риэлторов
PrivatePool - Наличие собственного бассейна
MlsId - id для риэлторов

target - Цена(Целевая переменная)
